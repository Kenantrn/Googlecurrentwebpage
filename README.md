# Google Current Web Page
Bu proje, Google'ın güncel ana sayfa tasarımını HTML ve CSS kullanarak yeniden oluşturuldu.

## İçerik

- HTML5
- CSS3
- Responsive değildir!!!

## Nasıl Çalıştırılır?

1. Projeyi bilgisayarınıza indirin.
2. index.html dosyasını bir tarayıcıda açın.
